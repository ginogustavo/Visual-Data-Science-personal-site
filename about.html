About this project

  • Using the provided sources SlateGunDeaths.CSV, I applied filtering and grouping to summary the records in the needed format. Initially global info By states and the frequency in order to separate Males and Females victims. This was accomplished using Python - Google Colab.
  • With the base files, I plot the US Map with neutral color codes for global information. Used as base the Samples in Observable web page.
  • I separated Totals of victims in relation to the whole country in order to plot the Scalar Bar at the bottom and code color the States polygons.
  • The other information is the death, gender, and age distribution per each State. displayed in a tooltip when the user hovers the State on the map. Groped by Gender and Ages.
  • The heaviest computing was the City frequency, where I had to draw circles according to the number of occurrences within a state plus the location using the latitude and longitude provided in the source file.
  • Finally, the frequency of victims by gender with respect to the whole country summarizes information about where is more dangerous for a specific gender. Plus visualizing the total number of only that gender per state. I happened to change the color code based on user changes on the dropdown.
